# gulpDemo
学习使用gulp；监视文件改动，同步更新到页面；自动编译es5,sass；js代码uglify
## 压缩js
```bash
gulp minify
```

## 转换ES5
```bash
gulp toes5
```

## 编译sass
```bash
gulp sass
```

## 压缩图片
```bash
gulp imagemin || npm run imagemin
```

## 监视文件改动，同步更新到页面；自动编译es5,sass；js代码uglify;压缩图片
```bash
npm run dev || gulp default
```
